# Blinkit Sales Analytics Dashboard – Power BI Project

 ----

## Project Overview
This project is an interactive Power BI dashboard created using a Blinkit grocery dataset to analyze sales performance, outlet trends, customer ratings, and product insights.

 ----

## Objectives
- Analyze total sales and average sales performance
- Understand outlet establishment trends
- Compare outlet sizes and outlet locations
- Analyze item types and fat content distribution
- Evaluate customer ratings and item visibility
  
 ----

## Tools & Technologies Used
- Power BI
- Power Query
- Data Cleaning
- Data Visualization
- Business Analytics

 ----

## Key Dashboard Features
- Interactive filters and slicers ( which Includes outlet location , outlet size outlet type ), 
- KPI cards for sales and ratings ( which includes total sales , average sales , no. of items , average rating )
- Sales trend analysis
- Outlet comparison visuals
- Product category insights
- Customer rating analysis

---

## Key Insights
- Tier 3 outlets generated the highest sales performance  ( 472k out of 1.2M )
- Medium outlet sizes contributed significantly to total sales 
- Certain item categories showed stronger customer engagement ( like fruits and vegetables and snack foods )
- Sales changed differently for outlets established in different years.
